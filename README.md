# Maze Runner

Game sederhana menggunakan HTML, CSS, dan JavaScript.

## Fitur

- Player berdenyut dengan efek `Math.sin`
- Kunci berkedip dengan `opacity` dinamis
- Suara bump saat player menabrak dinding
- Glow biru pada player
- Pintu berubah dari abu-abu ke hijau setelah semua kunci diambil
- Pause / resume dengan tombol `P`

## Kontrol

- `W`, `A`, `S`, `D` atau panah untuk bergerak
- `P` untuk pause / resume
- `N` untuk lanjut level setelah menyelesaikan level

## Menjalankan proyek

1. Install dependency:

```bash
npm install
```

2. Jalankan server:

```bash
npm start
```

3. Buka `http://localhost:8080`

## Packaging

```bash
npm run package
```
